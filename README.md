# gnomearch
instalacion de xorg y gnome

descarga con git

git clone git://github.com/abdiasriver/gnomearch

o con wget

wget https://github.com/abdiasriver/gnomearch/tarball/master -O -| tar xz



da permisos de ejecucion a los archivos con


cd gnomearch

chmod +x *

ejecuta con

./goinstall.sh

sudo reboot
